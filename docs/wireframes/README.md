# ワイヤーフレーム

投票ボードゲームの画面設計ワイヤーフレームです。

## ファイル一覧

### MVP対象画面

1. **01-top-page.drawio** - トップページ（対局一覧）
   - 進行中/終了の対局一覧
   - 対局カードのグリッド表示
   - 盤面サムネイル、ターン数、参加者数、投票締切

2. **02-login.drawio** - ログイン画面
   - メールアドレス・パスワード入力
   - パスワードリセットリンク
   - 登録リンク

3. **03-register.drawio** - ユーザー登録画面
   - ユーザー名・メールアドレス・パスワード入力
   - バリデーション表示

4. **04-password-reset.drawio** - パスワードリセット画面
   - メールアドレス入力
   - リセットメール送信

5. **05-profile.drawio** - プロフィール画面
   - アイコン画像表示
   - ユーザー名編集
   - 参加した対局一覧

6. **06-game-create.drawio** - 対局作成画面
   - ゲーム種類選択（MVP: オセロのみ）
   - 対局タイプ選択（MVP: AI vs 集合知のみ）
   - 先手選択

7. **07-game-detail.drawio** - 対局詳細画面
   - 盤面表示（8x8グリッド）
   - 手履歴
   - AI解説
   - 次の一手候補リスト
   - 投票機能

8. **08-candidate-post.drawio** - 候補投稿画面
   - 盤面上で手の位置を選択
   - 説明文入力（200文字制限）
   - プレビュー・投稿

9. **09-candidate-detail.drawio** - 候補詳細画面
   - 候補反映後の盤面プレビュー
   - 説明文表示
   - 投稿者・投票数表示
   - 投票機能
   - SNSシェア用

## 使い方

### draw.ioで開く

1. [draw.io](https://app.diagrams.net/) にアクセス
2. 「Open Existing Diagram」を選択
3. 各 `.drawio` ファイルを開く

### VSCodeで開く（推奨）

1. Draw.io Integration 拡張機能をインストール
2. `.drawio` ファイルをVSCodeで直接開く
3. 編集・保存が可能

## デザインシステム

### カラーパレット

- **プライマリ**: `#2563eb` (青)
- **セカンダリ**: `#10b981` (緑)
- **背景**: `#f9fafb` (グレー)
- **テキスト**: `#111827` (黒)
- **ボーダー**: `#e5e7eb` (薄いグレー)
- **無効**: `#9ca3af` (グレー)

### タイポグラフィ

- **大見出し**: 28-32px, Bold
- **中見出し**: 20-24px, Bold
- **小見出し**: 18px, Bold
- **本文**: 14-16px, Regular
- **キャプション**: 12-14px, Regular

### ボタン

- **プライマリ**: 青背景 (`#2563eb`)、白文字、角丸
- **セカンダリ**: 白背景、青文字、青ボーダー、角丸
- **成功**: 緑背景 (`#10b981`)、白文字、角丸
- **サイズ**: 高さ 35-50px、パディング 左右 20-40px

### カード

- **背景**: 白 (`#ffffff`)
- **ボーダー**: 薄いグレー (`#e5e7eb`)
- **影**: `shadow-sm`
- **角丸**: `rounded-lg` (8px)

### 盤面

- **グリッド**: 8x8
- **セルサイズ**: 40px (デスクトップ), 30px (モバイル)
- **背景**: 緑 (`#10b981`)
- **グリッド線**: 黒
- **黒石**: 黒円
- **白石**: 白円

## レスポンシブデザイン

### ブレークポイント

- **モバイル**: `< 768px`
- **タブレット**: `768px - 1024px`
- **デスクトップ**: `> 1024px`
