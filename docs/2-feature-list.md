# 機能一覧

## 1. ユーザー管理機能

### 1.1 ユーザー登録・認証

- ユーザー登録（メールアドレス + パスワード）
- ログイン / ログアウト
- パスワードリセット
- **MVP対象**: ✅

### 1.2 ソーシャルログイン

- Google / Twitter / GitHub 連携
- **MVP対象**: ❌（将来実装）

### 1.3 プロフィール管理

- ユーザー名の設定・変更
- アイコン画像の設定
- **MVP対象**: ✅（最小限）

## 2. 対局管理機能

### 2.1 対局一覧表示

- 進行中の対局一覧
- 対局の基本情報表示（ゲーム種類、ターン数、参加者数）
- **MVP対象**: ✅

### 2.2 対局詳細表示

- 現在の盤面表示
- 対局履歴（過去の手の一覧）
- 次の一手の投票状況
- AI による対局解説
- **MVP対象**: ✅

### 2.3 対局作成

- 新規対局の作成
- ゲーム種類の選択（オセロ）
- 対局設定（AI vs 集合知）
- **MVP対象**: ✅（オセロのみ）

### 2.4 対局終了

- 勝敗判定
- 対局結果の表示
- **MVP対象**: ✅

### 2.5 複数ゲーム対応

- チェス、囲碁、将棋の対局
- **MVP対象**: ❌（将来実装）

### 2.6 集合知 vs 集合知

- 2つのチームに分かれた対局
- **MVP対象**: ❌（将来実装）

## 3. 投票機能

### 3.1 次の一手候補の表示

- AI が生成した候補の表示
- ユーザーが投稿した候補の表示
- 候補の説明文表示（200文字以内）
- **MVP対象**: ✅

### 3.2 候補への投票

- 1ユーザー1票
- 投票の変更（締切前まで）
- 投票状況の表示（投票数）
- **MVP対象**: ✅

### 3.3 候補の投稿

- ユーザーによる次の一手候補の投稿
- 候補の説明文入力（200文字以内）
- 盤面上での手の選択
- **MVP対象**: ✅

### 3.4 投票締切・集計

- 日次バッチによる自動締切（JST 0:00）
- 投票結果の集計
- 最多得票の候補を次の一手として確定
- **MVP対象**: ✅

### 3.5 リアルタイム投票状況

- WebSocket による投票数のリアルタイム更新
- **MVP対象**: ❌（実装しない）

## 4. AI 機能

### 4.1 次の一手候補の生成

- Amazon Bedrock (Nova Pro) による候補生成
- 複数候補の生成（3〜5個程度）
- 各候補の説明文生成
- **MVP対象**: ✅

### 4.2 対局解説の生成

- 現在の盤面状況の解説
- これまでの対局の流れの解説
- **MVP対象**: ✅

### 4.3 AI の手の実行

- AI vs 集合知での AI の手の自動実行
- **MVP対象**: ✅

## 5. シェア機能

### 5.1 対局のシェア

- 対局詳細ページの URL シェア
- OGP 画像の生成（盤面のサムネイル）
- Twitter / Facebook / LINE シェアボタン
- **MVP対象**: ✅

### 5.2 特定ターンのシェア

- 過去の盤面状態の URL シェア
- 特定ターンの OGP 画像生成
- **MVP対象**: ✅

### 5.3 候補のシェア

- 次の一手候補の URL シェア
- 候補の OGP 画像生成
- **MVP対象**: ✅

## 6. 知識習得機能

### 6.1 専門用語リンク

- 候補説明文内の専門用語の自動リンク化
- **MVP対象**: ❌（将来実装）

### 6.2 専門用語解説ページ

- 専門用語の詳細解説
- 関連用語へのリンク
- **MVP対象**: ❌（将来実装）

## 7. 通知機能

### 7.1 投票開始通知

- 次の一手の投票が開始されたことを通知
- **MVP対象**: ❌（将来実装）

### 7.2 投票締切通知

- 投票締切が近づいたことを通知
- **MVP対象**: ❌（将来実装）

### 7.3 対局終了通知

- 参加していた対局が終了したことを通知
- **MVP対象**: ❌（将来実装）

## 8. 管理機能

### 8.1 対局の管理

- **MVP対象**: ❌（将来実装）

### 8.2 ユーザーの管理

- ユーザーの停止・削除
- **MVP対象**: ❌（将来実装）

## MVP 機能サマリー

### 実装対象（✅）

1. ユーザー登録・認証・プロフィール管理（最小限）
2. 対局一覧・詳細表示・作成・終了（オセロのみ）
3. 次の一手候補の表示・投票・投稿
4. 投票締切・集計（日次バッチ）
5. AI による候補生成・対局解説・手の実行
6. 対局・特定ターン・候補のシェア（OGP対応）

### 将来実装（❌）

1. ソーシャルログイン
2. 複数ゲーム対応（チェス、囲碁、将棋）
3. 集合知 vs 集合知
4. 専門用語リンク・解説ページ
5. 通知機能
6. 管理機能

### 実装しない機能

- リアルタイム投票状況（WebSocket）
