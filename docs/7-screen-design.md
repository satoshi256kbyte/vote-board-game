# 画面設計（ワイヤーフレーム）

## 画面一覧

### MVP対象画面

1. トップページ（対局一覧）
2. ログイン画面
3. ユーザー登録画面
4. パスワードリセット画面
5. プロフィール画面
6. 対局作成画面
7. 対局詳細画面
8. 候補投稿画面
9. 候補詳細画面

## 共通要素

### ヘッダー

```txt
┌─────────────────────────────────────────────────────────┐
│ [ロゴ] 投票対局              [対局作成] [プロフィール] │
└─────────────────────────────────────────────────────────┘
```

- ロゴ: クリックでトップページへ
- 対局作成ボタン: `/games/new` へ遷移（要認証）
- プロフィールアイコン: クリックでドロップダウン
  - プロフィール
  - ログアウト
- 未認証時: [ログイン] [登録] ボタン表示

### フッター

```txt
┌─────────────────────────────────────────────────────────┐
│                © 2026 投票対局                         │
└─────────────────────────────────────────────────────────┘
```

- 対局終了: 勝敗表示、投票不可

### レスポンシブ

- モバイル: 左右カラムを縦に配置

## デザインシステム

### カラーパレット

- プライマリ: `#2563eb` (青)
- セカンダリ: `#10b981` (緑)
- 背景: `#f9fafb` (グレー)
- テキスト: `#111827` (黒)
- ボーダー: `#e5e7eb` (薄いグレー)

### タイポグラフィ

- 見出し: `font-bold text-2xl`
- 本文: `text-base`
- 小文字: `text-sm`

### ボタン

- プライマリ: 青背景、白文字
- セカンダリ: 白背景、青文字、青ボーダー
- サイズ: `px-4 py-2 rounded-md`

### カード

- 背景: 白
- ボーダー: 薄いグレー
- 影: `shadow-sm`
- 角丸: `rounded-lg`

### 盤面

- グリッド: 8x8
- セルサイズ: 40px (デスクトップ), 30px (モバイル)
- 背景: 緑 (`#10b981`)
- グリッド線: 黒
- 黒石: 黒円
- 白石: 白円

## アクセシビリティ

- キーボード操作対応
- スクリーンリーダー対応
- 適切なコントラスト比
- フォーカス表示
- エラーメッセージの明確化

## レスポンシブデザイン

### ブレークポイント

- モバイル: `< 768px`
- タブレット: `768px - 1024px`
- デスクトップ: `> 1024px`

### モバイル対応

- ハンバーガーメニュー
- 縦スクロール
- タッチ操作最適化
- 盤面サイズ調整
