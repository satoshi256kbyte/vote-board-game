# Error Handling Tests

このディレクトリには、エラーシナリオとエッジケースに関するE2Eテストが含まれます。

## テストファイル

- `network-errors.spec.ts` - ネットワークエラーのテスト
- `session-timeout.spec.ts` - セッションタイムアウトのテスト
- `validation-errors.spec.ts` - バリデーションエラーのテスト

## カバレッジ

- ネットワーク障害時のエラーメッセージ
- セッションタイムアウト時のリダイレクト
- 必須フィールドのバリデーション
- 存在しないゲームへのアクセス
- 404エラーページの表示
- 投票期間終了後のエラー
