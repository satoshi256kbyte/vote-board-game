# Voting Flow Tests

このディレクトリには、投票機能に関するE2Eテストが含まれます。

## テストファイル

- `vote-submission.spec.ts` - 投票送信のテスト
- `candidate-submission.spec.ts` - 候補投稿のテスト
- `vote-validation.spec.ts` - 投票バリデーションのテスト

## カバレッジ

- 次の一手候補の表示
- 候補の選択と投票
- 投票成功の確認
- 重複投票の防止
- 投票期間終了後のエラー
- 新しい候補の投稿
- 候補の説明文表示
- 無効なデータのバリデーション
